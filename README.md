# WP Reboot Wordpress Theme
This wordpress theme features a Dropdown Menu, 2 Column Layout, BootStrap CSS Framework, Fontawesome Icons

## Functions.php
The wordpress functions.php has a various code snippets to improve the theme functionality.

## Templates
There is one Custom Template - ALLPAGES (that is one column - centered in the main container)
